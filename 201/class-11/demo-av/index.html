Audo Video Elements

Audio

Common attributes of audio

  • autoplay: Boolean; use sparingly (if at all)
  • controls: provides playback controls to the user
  • loop: Boolean; will play audio over and over (and over)
  • muted: Boolean; can use this to mute audio on page load
  • src: link to the audio file
  • volume: Range of 0.0 (silent) to 1.0 (loud)

Video

  • autoplay: Boolean; use sparingly (if at all)
  • controls: provides playback controls to the user
  • height: MUST be an absolute value (no percentages)
  • muted: Boolean; can use this to mute audio on page load
  • poster: user an alternate image for the thumbnail (default is first fream of video)
  • src: link to the video file
  • volume: Range of 0.0 (silent) to 1.0 (loud)
  • width: May be an absolute value or percentages
