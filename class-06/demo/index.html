Variables

We can store data inside variables (like buckets).

Data Types

There are a bunch of different data types in programming languages. The three we care about today are:

  • strings - "this is a string"
  • numbers - 420
  • boolean - either true or false

Concatenation

Putting 2 strings together, to make 1 string

"Hello" + "World"

Order of execution

Where we place our script tag matters, if we place it in the head, it will run before the page loads.

If we place it in the body, it will run, where we put it (which is what this example does).
